## 🎯 **使用指南**

### **开发模式**
```bash
npm run dev    # 开发模式 (热重载)
npm run build  # 生产构建
npm test       # 运行测试
```

### **扩展安装**
1. 运行 `npm run build`
2. 打开 `chrome://extensions/`
3. 启用开发者模式
4. 点击"加载已解压的扩展程序"
5. 选择项目根目录
